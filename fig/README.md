# Figure Folder

This folder documents the whole project visually.

Use the numbered subfolders in pipeline order:

- `01_dataset`
- `02_preprocessing`
- `03_patches`
- `04_classical`
- `05_autoencoder`
- `06_results`

Suggested naming pattern:

```text
<step>_<category>_<short_description>.png
```

Examples:

- `dataset_bagel_examples.png`
- `preprocessing_carrot_normalization.png`
- `patches_cable_gland_grid.png`
- `classical_bagel_score_map.png`
- `autoencoder_bagel_reconstruction.png`
- `results_comparison_panel.png`

These figures should be usable for debugging, the final report, and quick repository review.
